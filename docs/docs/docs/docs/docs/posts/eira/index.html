Descripción

la tayra (Eira barbara), también llamada viejo de monte, irará, eirá, tolomuco, cabeza de mate, cabeza de viejo, manco, sanjool, hurón mayor o comadreja es una especie de mamífero carnívoro de la familia Mustelidae. Habita en selvas tropicales desde las planicies costeras de México hasta el norte de Argentina. Mide alrededor de 70 cm y pesa 5 kg. Tiene la cabeza blanquecina o amarilla y es de hábitos arborícolas. Es omnívoro, se alimenta de frutos, insectos, carroña y pequeños vertebrados.
